# EnterpriseFlow
EnterpriseFlow is a full-stack company management system with employees, departments and projects. Includes SQL database, REST API, and responsive web UI with advanced filtering, sorting, pagination and CRUD operations.
# CorpManager

A complete company management system with employees, departments, projects and addresses management. Built with SQL database, REST API and modern web interface.

![Version](https://img.shields.io/badge/version-1.0.0-blue)
![License](https://img.shields.io/badge/license-MIT-green)

## ✨ Features

- **Employee Management** - Full CRUD operations with filtering, sorting and pagination
- **Departments Overview** - View department statistics and managers
- **Projects Tracking** - Monitor active and completed projects  
- **Advanced Search** - Filter by name, job title, city, department and salary range
- **Responsive Design** - Works perfectly on desktop, tablet and mobile
- **Modern UI** - Beautiful gradients, smooth animations and intuitive interface
- **SQL Solutions** - Built-in SQL query examples for learning

## 🛠️ Technologies

- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Backend:** Node.js, Express.js
- **Database:** MySQL
- **API:** RESTful architecture
- **Icons:** Font Awesome 6
